## プロジェクト概要

■ 概要

本プロジェクトは、コンボシステムや必殺技を搭載した3Dアクションゲームとして開発した。
特に「戦闘の爽快感」と「シネマティックなカメラ演出」の設計に重点を置いている。

■ 設計（コア部分）

・戦闘・コンボシステム（最重要）

敵へのターゲット・吸い付き処理や、連続攻撃を管理するコンボシステム（自己修復機能付き）を構築。
攻撃のヒット判定を細かく調整し、アクションゲームとしての手触りを向上させた。

・動的UIシステム

プレイヤーの状況に応じた初回解放通知や、敵との距離に応じて表示・非表示が切り替わる頭上HPバーなど、動的なUI制御システムを設計した。

■ 実装（ゲームプレイ）

・プレイヤー・敵アクション

抜刀・納刀アニメーションの制御や、敵撃破時のフェードアウト消失処理などを実装。

・システム連動

リセット機能や、特定の通知と連動したテレポート機能など、ゲーム進行を補助するシステムを組み込んだ。

■ 演出・体験設計

・ダイナミックなカメラ制御

必殺技発動時のシネマティックなカメラワークやボスカメラを実装し、戦闘の迫力を大幅に強化した。

・チュートリアルと導線

特定エリア到達時のテキスト表示や、チュートリアル専用UIを配置し、プレイヤーの学習体験をサポート。

■ 工夫・改善点

・戦闘ロジックとカメラ演出を強く連動させることで、没入感の高いアクションを実現した

・敵のHPバー表示を距離判定で制御し、画面のUI情報の煩雑さを軽減した

・複数人開発において、アセット配置とシステム実装の役割を明確化し、コンポーネントごとの連携を意識した設計を行った

## 開発メンバーと担当箇所

本プロジェクトの開発に貢献したメンバーと、それぞれの主な実装箇所は以下の通りです。

### sogotoya
* 戦闘システムの実装(コンボシステム、ヒット判定の修正、敵へのターゲット・吸い付き処理など)
* 必殺技と演出の作成(抜刀・納刀アニメーションの調整、必殺技発動時のシネマティックカメラワーク、敵撃破時のフェードアウト演出)
* UI機能の実装(チュートリアルUI、特定エリアでのテキスト表示、敵との距離に応じた頭上HPバー表示、各種メニュー画面の整備)
* その他システム機能(リセット機能、初回解放通知とテレポートの連携、ボスカメラやタイトルカメラの制御)

### Abubu
* プロジェクトの初期構築(パッケージ設定、初期プロジェクトデータの追加)
* アセットとマップの組み込み(地形(Terrain)とコライダーの調整、マップ・ツールアセットの導入)
* キャラクターモデルの設定(プレイヤーモデルの追加)
* タイトル画面の基礎作成
